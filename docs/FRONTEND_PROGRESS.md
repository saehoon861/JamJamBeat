    { "x": 0.12, "y": 0.34, "z": -0.02 },
    // ... 21개 랜드마크

```javascript
{
  "label": "Fist",
  "confidence": 0.91,
  "probs": [0.01, 0.91, 0.03, 0.02, 0.01, 0.01, 0.01],
  "model_version": "v1"
}
```

```javascript
// Alias → Canonical label 변환
"class1", "isfist" → "Fist"
"class2", "paper", "open_palm" → "OpenPalm"
"isv", "class3" → "V"
"class4", "ispinky" → "Pinky"
"class5", "isanimal" → "Animal"
"class6", "is_k_heart", "k-heart" → "KHeart"
```

#### 안정화 파라미터
