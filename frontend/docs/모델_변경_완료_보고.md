## ⚠️ 알려진 이슈 및 제한사항

### 1. Warmup 지연
- **현상**: 손을 올린 후 0.5초 대기
- **원인**: 8프레임 수집 필요
- **영향**: 게임 시작 시 한 번만 발생
- **해결**: 불가 (모델 설계 상 필수)

### 2. 손 사라짐 시 버퍼 리셋
- **현상**: 손이 카메라에서 사라지면 버퍼 초기화
- **원인**: 상태 일관성 유지
- **영향**: 다시 손 올릴 때 warmup 반복
- **해결**: 손을 계속 올려두고 플레이

### 3. 저사양 기기 성능
- **현상**: 구형 모바일에서 느릴 수 있음
- **해결**: `?seqModel=0`으로 기존 모델 사용

### 4. 모델 크기 증가
- **현상**: 1.2 MB → 초기 로딩 증가
- **영향**: 4G/5G에서는 무시 가능
- **해결**: 없음 (압축 불가)

```javascript
const USE_SEQUENCE_MODEL = (() => {
  const params = new URLSearchParams(window.location.search);
```
