Drop MP3 files in this folder to enable sample-based playback.

Required filenames:
- 드럼.mp3 (preferred)
- drum.mp3 (fallback)
- drum-mushroom.mp3 (fallback)
- 피리.mp3 (preferred: OpenPalm -> xylophone object)
- whistle.mp3 (fallback)
- xylophone.mp3 (fallback)
- xylophone-vine.mp3 (fallback)
- 피아노.mp3 (preferred: V -> tambourine object)
- piano.mp3 (fallback)
- tambourine.mp3 (fallback)
- tambourine-flower.mp3 (fallback)
- 심벌즈.mp3 (preferred: KHeart -> fern object)
- cymbals.mp3 (fallback)
- fern.mp3 (fallback)
- fist-beat.mp3
- pinky-chime.mp3 (fallback)
- 고양이.mp3 (Animal random)
- 강아지.mp3 (Animal random)
- cat.mp3 (Animal random fallback)
- dog.mp3 (Animal random fallback)
- owl.mp3 (fallback)
- heart-bloom.mp3
- animal-roll.mp3

If a file is missing, the app falls back to synthesized Web Audio sound for that action.
