# poc/

Proof of Concept을 위한 **간단한 실험 코드**가 있는 폴더입니다.

여기 코드는 최종 시스템 코드가 아니라  
아이디어가 가능한지 빠르게 검증하기 위한 코드입니다.

---

## 1. mediapipe_test.py
**역할:** MediaPipe Hands가 정상적으로 손 랜드마크를 추출하는지 테스트합니다.

웹캠 또는 영상 입력을 사용하여  
손의 21개 랜드마크가 제대로 검출되는지 확인합니다.

---

## 2. landmark_visualization.py
**역할:** 추출된 랜드마크를 화면에 시각적으로 표시합니다.

랜드마크 위치가 정확한지 확인하고  
모델 학습에 사용할 수 있는 데이터인지 검증하는 용도입니다.

---

## 3. mlp_gesture_test.py
**역할:** 간단한 MLP 모델을 이용하여 제스처 분류를 테스트합니다.

Transformer 모델을 사용하기 전에  
baseline 모델의 성능을 확인하기 위한 코드입니다.

---

## 4. dataset_builder.py
**역할:** MediaPipe로 추출한 랜드마크 데이터를  
모델 학습에 사용할 수 있는 **dataset 형태로 변환**합니다.
